## Description

The purpose of this code is to read a file with comma seperated names. The program will then process the names and output either:
* a list of alphabetically sorted names
* a list of names sorted by length
* a file with names and the length of each name in dictionary-like fashion

The user should specify an output file where the names are written to. The output will be one name pr. line of the file.
